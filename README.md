# Neural Networks from Scratch

This project is a from-scratch implementation of neural networks and backpropagation, built to deeply understand how modern machine learning models work.

Instead of relying on libraries like PyTorch or TensorFlow, all core components (autodiff, gradients, training loops) are implemented manually.


---

## Goals

- Understand backpropagation from first principles
- Implement a minimal autodiff engine
- Build neural networks without external ML frameworks
- Train simple models (regression, classification, language modeling)

---
